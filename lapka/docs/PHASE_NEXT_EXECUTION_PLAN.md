## Execution Strategy (Self Plan)

### Track A: PMF + ROI Hardening (0-6 months)

Goal: prove measurable clinic ROI on real operations.

1. Lock KPI contracts used by platform dashboards and exports.
2. Ensure every KPI has owner/source/frequency documented.
3. Standardize weekly ROI report pack for pilot clinics.
4. Add acceptance gates for ROI regressions in release checks.

### Track B: Repeatable Growth System (6-18 months)

Goal: make onboarding/sales/support reproducible for clinic chains.

1. Standardize enterprise onboarding runbook (roles, legal, security, rollout).
2. Normalize platform terminology and UX copy (RU/EN) in operational modules.
3. Add implementation scorecards for each new clinic chain rollout.
4. Define support SLA ladder and escalation matrix.

### Track C: Category Leadership Foundations (18+ months)

Goal: increase switching cost through deep operational integration.

1. Formalize AI moat assets registry (datasets, prompts, eval suites, gates).
2. Define integration roadmap (LIS/PACS/payments/insurance APIs).
3. Add ecosystem partner model and certification criteria.
4. Define moat health metrics (coverage, quality, and time-to-improvement).
