# Outcome Metrics Tree

## North Star

Clinics complete more safe visits per day with lower operational loss.

## Driver Tree

- Throughput
  - Visits completed/day
  - Average wait time
- Reliability
  - No-show rate
  - Follow-up completion
- Quality and safety
  - Protocol completeness
  - Policy-violation rate in AI workflows
- Financial impact
  - Revenue per branch
  - Cost per completed visit

## Measurement Rules

- Every metric has baseline and post-rollout period.
- Compare at clinic and branch levels.
- Store confidence notes for each KPI (strong/medium/weak evidence).
