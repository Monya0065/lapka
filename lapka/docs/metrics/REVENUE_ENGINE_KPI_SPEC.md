# Revenue Engine KPI Spec

## Purpose

Define revenue KPIs with formulas and owners.

## Primary KPIs

- Lead to Demo Conversion = demos / qualified leads
- Demo to Pilot Conversion = pilots / demos
- Pilot to Paid Conversion = paid clinics / pilots
- ARPA = MRR / active paid clinics
- Net Revenue Retention (NRR)
- CAC Payback (months)

## Data Sources

- CRM pipeline exports
- Platform billing APIs
- Clinic activation events

## Reporting Cadence

- Weekly operating dashboard
- Monthly executive dashboard

## Targets (initial)

- Demo to Pilot >= 35%
- Pilot to Paid >= 55%
- CAC payback <= 9 months
- NRR >= 110%
