# GTM Operating System

## Scope

Operational GTM baseline for Phase 2 execution across clinic chains.

## ICP

- Multi-branch veterinary clinics (2-20 branches)
- Roles with clear ROI owner: clinic admin / network admin
- Existing digital scheduling and document flow

## Positioning

- Lapka is an operations + clinical workflow system with governed AI.
- Primary value: faster throughput, lower no-show impact, safer documentation quality.

## Funnel Stages

1. Qualified lead
2. Discovery
3. Demo
4. Pilot signed
5. Activation (first branch live)
6. Expansion (additional branches)
7. Renewal

## Weekly Cadence

- Monday: pipeline review
- Wednesday: pilot health review
- Friday: revenue + retention review

## Owners

- GTM owner: Head of Growth
- Product owner: Product Lead
- Technical owner: Engineering Lead
- Customer success owner: CS Lead
