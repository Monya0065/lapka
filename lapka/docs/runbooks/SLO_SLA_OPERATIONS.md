# SLO/SLA Operations

## Service Levels

- API availability SLO: 99.9%
- P95 latency SLO (critical endpoints): <= 600ms
- Error budget policy: monthly

## SLA Tiers

- Standard: business-hours support
- Enterprise: 24/7 incident response + faster acknowledgment

## Operational Process

1. Monitor SLI dashboards
2. Detect burn-rate breaches
3. Trigger incident runbook if needed
4. Apply mitigation and track recovery

## Core Indicators

- Availability
- Latency
- Error rate
- Queue backlogs
- Background job failures

## Review Cadence

- Weekly SLO review
- Monthly SLA report
- Quarterly threshold recalibration
